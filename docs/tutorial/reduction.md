# Basic Reduction

The `trired` command provides an interface to apply basic reduction of obtained `OBJECT` frames. The basic usage is as follows. `DATABASE` is the database fits file generated by `tripre`. `INPUT` and `OUTPUT` are the FITS file to be reduced and the output filename, respectively.

``` console
$ trired DATABASE INPUT OUTPUT
```

The following operations are applied:

1. A photo-sensitive area is extracted. Thus, peripheral reference pixels are trimmed out. Currently, the bias correction by the reference pixels is not implemented yet.
1. Dark current is subtracted. The appropriate dark frame is extracted from the database. The same dark frame is subtracted from each frame.
2. Flat-frame calibration is applied. The appropriate flat frame is extracted from the database. Each frame is divided by the same flat frame.

The input file is expected to be a FITS data cube. The output file is the reference-trimmed, dark-subtracted, and flat-corrected data cube.

There are two options to obtain the result as a 2-dimensional image. The `--ql` option is used to generate a quick-look image. The mean values along with `NAXIS3` are calculated. The `--stack` option generates a 3&sigma;-clipped mean image instead of a simple average.
