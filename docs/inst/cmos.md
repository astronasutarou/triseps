# CMOS sensor

## Specifications
Specifications of the CMOS sensors are summrized below.

|Parameter|Value|
|---|---|
|Pixel Size|19 &mu;m|
|Sensor Dimension|2220&times;1360|
|Effective Area|2160&times;1280|
|ADC bit depth|14 bit (data are stored as 16 bit)|
|Max data rate|8 TB/night/CMOS|
|Max frame rate|98 fps for full-frame images|
|Bias level|&sim;1000 ADU|
|Linearity|&sim;1% or better for <12,000 ADU|
|Saturation|16383 ADU or 72000e⁻|
|Timing accuracy|&sim;0.5 msec (synchronized with a GPS clock)|
